Optical Sizing

One of the things Mike likes about Type is that Scaling type to different sizes allows her to adjust the order of attention her Type gets from the reader and helps her readers, among other things, navigate from the shorter headlines to the longer text they choose to read.

Optical sizing: a primer

The idea of optical sizing originated in metal type. This was a practice of altering the letterforms and stroke weights based on the physical size of the type. Smaller type was a bit more open, the stroke weights a bit sturdier. All in order to enhance legibility at small sizes without sacrificing the design details at larger ones.

Big and small

But too often for Mike, when she tries to scale the type too large, it becomes “...too horsey, like too large a pet in too small a space.” And if she tries to scale too small, Mike finds the type too fragile, like “...trying to ride a long way on a cat.”

Lucky for Mike, she found out that type lovers like her can scale their type using OpenType Variable Fonts technology optical size axis (now appearing on most Operating Systems and browswers).

Now Mike’s Type looks smooth and consistent, the style of each size appearing visually suited with the others, larger,sizes need less,space and smaller sizes are more readable.

Optical Sizing in action

Each time the user changes the point size of their type, either the program they are using will select the matching optical point size, for them

Or using a Variable font menu, the user may select an “opsz” in points, per size

Or writing CSS, a user may specify “opsz” the same way as weight of width to get the optical size desired.
